# 📧 Django Email System

## 📌 Description
This project demonstrates how to send emails using Django.  
It includes basic email configuration and functionality for sending emails from a Django application.

---

## 🚀 Features
- 📩 Send emails using Django
- ⚙️ SMTP configuration
- 📬 Email integration in backend
- 🔒 Secure email handling

---

## 🛠️ Tech Stack
- Python
- Django
- SMTP (Email Service)
